Sales Data Analysis Project

Project Overview

This project analyzes sales performance data using SQL and Power BI to uncover key business insights such as customer behavior, product performance, and revenue trends.

The goal is to simulate a real-world business scenario where data is used to support decision-making.

---

Business Objectives

* Identify top-performing products and categories
* Analyze customer purchasing behavior
* Track monthly and yearly sales trends
* Understand revenue contribution by region/customer segment
* Provide actionable insights for business growth

---

Tools & Technologies

* SQL (Data extraction, cleaning, analysis)
* Power BI (Data visualization & dashboards)
* Excel (Data preparation)
* GitHub (Version control & portfolio hosting)

---

## 📂 Project Structure

* `datasets/` → Raw CSV datasets
* `Sql/` → SQL scripts for analysis
* `powerbi/` → Power BI dashboard file
* `Screenshots/` → Dashboard images & visual outputs

---

Key Analysis Performed

* Total sales and revenue analysis
* Customer segmentation using purchase patterns
* Product ranking using SQL window functions
* Monthly sales trend analysis
* Top-performing regions and categories

---

Key Insights

* A small group of products contributes to most of the revenue
* Repeat customers generate significantly higher lifetime value
* Sales performance varies strongly by region and category
* Certain months show seasonal spikes in demand
